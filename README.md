## What I Work On

**Expected Metrics and Models**
xG, xA, expected threat (xT), Markov chain possession models, and match outcome prediction built from event data.

**Player Analysis and Recruitment Tools**
Similarity models, clustering, radar plots, and scouting pipelines that turn large player databases into defensible shortlists.

**Data Pipelines**
Automated systems that collect, store, and update football data from multiple providers. Built to run repeatedly, not just once.

**Visualisation and Dashboards**
Passing networks, heatmaps, pitch control, Voronoi diagrams, and interactive Streamlit applications that make football data accessible beyond the analyst.

**Tracking and Physical Data**
Off-ball movement, player positioning, and fitness metrics to understand what happens away from the ball.
